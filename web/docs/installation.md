# Installation

1. Download, and install [Python](https://www.python.org/) (_v3.10 or higher_)
2. In the terminal, run:

```bash
pip install hachi-machi --pre
```

3. Then run:

```bash
hxmx --version
```

If you see a banner in the console, it's been sucessfully installed.
