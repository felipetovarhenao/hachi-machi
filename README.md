# hachi machi

To learn more, please visit: https://felipe-tovar-henao.com/hachi-machi

## Setup

1. Install [conda](https://anaconda.org/):
2. In the terminal, run:

```sh
conda env create -f environment.yml
conda activate hxmx
```

## On each session

Run to activate:

```sh
conda activate hxmx
```

or to deactivate:

```sh
conda deactivate
```
